# Google Docs workflow

You'll keep your **weekly log** in a single Google Doc. That's it — no git, no
markdown, no commits. Open the doc, scroll to this week, write.

## :material-numeric-1-circle: Your one doc

The PI created a private Google Doc just for you. Find the link on **your
student page** (it's the big "Open my weekly log" button at the top).

Bookmark it. Use the same doc all summer. Add a new section each Friday —
don't make new docs every week.

## :material-numeric-2-circle: Suggested doc structure

```
Title: Changik Park — Brown Ocean Effect — Summer 2026 Weekly Log

[Research question]   ← refined in week 1
[Plan]                ← refined in week 1

[Week 1 — Jun 22 – Jun 26]
  Goal:
  Done:
  Blockers:
  Figures (insert images / links):
  Next week:
  Self-rating (0–100):

[Week 2 — Jun 29 – Jul 03]
  ... (same template)

[Week 3 ...]
```

A copy-paste template lives at **[Weekly log template](weekly-template.md)**.

## :material-numeric-3-circle: Getting PI feedback

The PI reads your doc over the weekend after Friday's update.

- Feedback comes as **Google Docs comments** (highlighted text → 💬 icon).
- Reply to comments in the doc — keep the discussion threaded there.
- Use `@mention` (`@hyunglokkim@...`) to flag the PI on a specific question.
- Use **Suggesting mode** (top-right pencil → Suggesting) when proposing
  changes to your own earlier text — keeps history clean.

## :material-numeric-4-circle: Sharing & permissions

- Your doc is shared with the PI as **Editor** and other interns as **Viewer**.
- Don't change sharing settings without asking — the PI dashboard depends on
  the link staying stable.
- Drop figures *into the doc* (Insert → Image → Upload). Don't link to files
  on your local drive — they won't load for anyone else.

## :material-help-circle-outline: Quick tips

- **Tables of numbers** → paste from a spreadsheet, Google Docs handles it well.
- **Code snippets** → use `Format → Code` or a monospace font block.
- **Math** → `Insert → Equation`. For complex equations, screenshot from LaTeX.
- **Citations** → footnote (`Insert → Footnote`) or a "References" section
  at the bottom of the doc.

!!! tip "Keep it honest, not pretty"
    A doc the PI can skim in 5 minutes is worth more than a polished doc that
    takes an hour to write. Bullets > paragraphs.
