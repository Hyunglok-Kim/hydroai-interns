# Weekly log template

Copy the block below into your Google Doc each Friday — one section per week,
in the same doc.

```
─────────────────────────────────────────────
Week N — short theme              [Month DD – Month DD, 2026]
─────────────────────────────────────────────

Status: on track / behind / pivoting

Goal for the week
  What were you trying to do? (one paragraph)

What got done
  • bullet
  • bullet

Blockers / open questions
  • bullet — flag the PI with @mention if urgent

Figures / artifacts
  (paste figures inline; one-line caption under each)

Next week's plan
  • bullet

Self-rated progress: __/100
  Short justification.
─────────────────────────────────────────────
```

## :material-format-list-bulleted-type: How to fill it well

- **Goal first.** If you can't write it in one sentence, that's the real
  blocker — bring it up at Monday stand-up.
- **Be honest in "what got done."** A debugged-but-not-finished pipeline is
  worth more than three vague bullets.
- **Show, don't tell.** A messy figure beats no figure. Paste it inline even
  if it's ugly.
- **Self-rating.** This isn't a grade — it's how you'd answer "are you on
  track to give the final talk?" 50% by week 4 is healthy. 50% by week 8 is
  a fire alarm.

## :material-image: How figures should look

- **Inline in the doc** (Insert → Image → Upload from computer).
- Caption underneath in italics: *"Fig. wk3-1 — SMAP L3 daily mean 2023, masked over CONUS."*
- Don't link to files on your laptop — they won't load for anyone else.
- For interactive plots, paste a screenshot AND link to the notebook on lab storage.

## :material-link-variant: Linking other resources

- Link Jupyter notebooks, slides, or larger figures stored on lab storage.
- For papers you read, prefer DOI links over Google Scholar URLs.
- If you reference another intern's work, link to their doc.
